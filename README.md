# TechRobotics-AR 🤖🌾

TechRobotics-AR is a student-led technology and robotics platform
focused on smart farming, automation, and AI-based solutions.

## 🚀 What We Do
- Smart Agriculture Robotics
- Arduino & IoT Projects
- Automation for Indian Farmers
- Future-ready Technology Solutions

## 📂 Projects Included
- Multipurpose Agriculture Robot
- Smart Dustbin Project
- Home Security Alert System
- Touchless Tap System

## 🌱 Vision
To use technology and robotics to solve real-world problems
and help Indian farmers and small businesses.
